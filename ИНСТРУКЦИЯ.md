# Как запустить бота — пошаговая инструкция

---

## ШАГ 1 — Создай бота в Telegram

1. Открой Telegram
2. В поиске найди **@BotFather** и открой его
3. Напиши ему: `/newbot`
4. Он спросит имя бота — напиши например: `Лечебное Голодание`
5. Он спросит username — напиши например: `moy_golodanie_bot`
6. BotFather пришлёт сообщение с **токеном** — это длинная строка вида:
   ```
   7812345678:AAFghIjkLMnopqRSTUvwxYZ123456789abc
   ```
   **Скопируй и сохрани этот токен.**

---

## ШАГ 2 — Создай файл .env

1. Перейди в папку `bot` (там где лежат файлы main.py и остальные)
2. Найди файл `.env.example`
3. **Скопируй** его и переименуй копию в `.env` (без слова "example")
4. Открой файл `.env` любым текстовым редактором (Блокнот, TextEdit и т.д.)
5. Замени текст `1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ` на **свой токен** из Шага 1
6. Сохрани файл

Должно получиться так:
```
BOT_TOKEN=7812345678:AAFghIjkLMnopqRSTUvwxYZ123456789abc
```

---

## ШАГ 3 — Скопируй фотографии

1. Найди папку `photos` в архиве Telegram (она уже есть в твоей папке `ChatExport_2026-06-02`)
2. Скопируй эту папку `photos` **внутрь папки `bot`**

После этого структура папок должна выглядеть так:
```
bot/
├── main.py
├── content.py
├── db.py
├── requirements.txt
├── .env          ← файл с токеном
└── photos/       ← папка с фотографиями
    ├── photo_1@...jpg
    ├── photo_2@...jpg
    └── ...
```

## ШАГ 4 — Установи Python (если не установлен)

### На Mac:
1. Открой браузер, перейди на https://www.python.org/downloads/
2. Нажми большую кнопку **Download Python**
3. Открой скачанный файл и установи Python (нажимай "Continue" → "Install")

### На Windows:
1. Перейди на https://www.python.org/downloads/
2. Скачай и запусти установщик
3. **ВАЖНО:** На первом экране поставь галочку **"Add Python to PATH"**
4. Нажми "Install Now"

---

## ШАГ 5 — Установи зависимости и запусти бота

### На Mac:
1. Открой приложение **Терминал** (найди через Spotlight: Cmd+Пробел, напиши "Терминал")
2. Напиши команду для перехода в папку бота (замени путь на свой):
   ```
   cd ~/Downloads/ChatExport_2026-06-02/bot
   ```
3. Установи нужные библиотеки:
   ```
   pip3 install -r requirements.txt
   ```
4. Запусти бота:
   ```
   python3 main.py
   ```

### На Windows:
1. Открой папку `bot` в Проводнике
2. В адресной строке папки напиши `cmd` и нажми Enter — откроется командная строка
3. Установи нужные библиотеки:
   ```
   pip install -r requirements.txt
   ```
4. Запусти бота:
   ```
   python main.py
   ```

## Готово! 🎉

Если в терминале появилось сообщение **"Бот запущен!"** — всё работает.

Открой Telegram, найди своего бота по username и напиши `/start`.

---

## Команды бота

| Команда | Что делает |
|---|---|
| `/start` | Начать курс с самого начала |
| `/continue` | Продолжить с того места где остановился |
| `/status` | Посмотреть на каком шаге ты сейчас |
| `/steps` | Список всех шагов курса |
| `/goto stage3_start` | Перейти напрямую на любой шаг |

---

## Важно — бот должен быть запущен

Пока терминал открыт и бот запущен — он работает.
Если закроешь терминал — бот выключится.

Чтобы снова запустить — повтори Шаг 5 (только пункты 2 и 4, устанавливать повторно не нужно).

---

## Если что-то пошло не так

**Ошибка "BOT_TOKEN не задан"** → проверь файл `.env`, там должна быть строка с токеном

**Ошибка "No module named aiogram"** → не установлены библиотеки, повтори команду `pip install -r requirements.txt`

**Фото не отправляются** → проверь что папка `photos` лежит внутри папки `bot`

**Бот не отвечает** → убедись что в терминале написано "Бот запущен!" и терминал не закрыт
